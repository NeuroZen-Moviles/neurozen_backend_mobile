#  NeuroZen - Backend

¡Bienvenido al repositorio de **NeuroZen**! Este proyecto está diseñado para ofrecer una solución integral de salud y bienestar mental a través de una aplicación móvil de alta fidelidad, respaldada por una arquitectura de backend robusta y escalable.
